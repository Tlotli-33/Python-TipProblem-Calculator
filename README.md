# Python-TipProblem-Calculator
Project Title

Python Tip Calculator

Description

This is a simple Python program that calculates how much tip to leave based on the total meal cost and the tip percentage entered by the user.

The program:

Accepts the meal price (example: $85.50)
Accepts the tip percentage (example: 15%)
Removes the $ and % symbols
Converts the values into numbers
Calculates the tip amount
Displays the final tip rounded to 2 decimal places
Features
User input handling
String slicing
Type conversion (str → float)
Percentage calculations
Formatted output using f-strings
Technologies Used
Python
